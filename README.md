# Sorting Simulator
This project includes code for the sorting simulator of Prime Vision.

## Dependecies
```
pip install sanic
pip install asyncio
```

## Getting started
```
git clone https://github.com/TimoThans33/sorting-sim.git
```
```
python3 main.py
```
To simulate the server side you can use the following:
```
python3 echo-server.py
```
